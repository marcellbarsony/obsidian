# Constants

```c
const int INT = 25;                  // Integer constant
const char CHAR = 'A';               // Character constant
const float FLOAT = 15.66;           // Floating point constant
const double DOUBLE = 123.456789;    // Double precision floating point constant
const int ARRAY[] = {1, 2, 3, 4, 5}; // Array constant

// Structure constant
struct Point {
    const int x;
    const int y;
};

const struct Point pt = {10, 20};
```
