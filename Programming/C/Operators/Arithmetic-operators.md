## Augmented assignment operators

Shorthand forms of arithmetic operators
```c
x+=1; // x = x + 1;
x-=1; // x = x - 1;
x*=1; // x = x * 1;
x/=1; // x = x / 1;
x%=1; // x = x % 1;
```

For exponentiation use the `pow()` function variants from `math.h`
