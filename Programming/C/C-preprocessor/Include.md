# `#include`

Before the program is compiled, it runs through preprocessing.
Header files can be split into 2 categories.
```c
// System-wide header files
#include <stdio.h>
#include <stdlib.h>
#include <math.h>

// Local header files
#include "myheader.h"
#include "mydir/myheader.h"
#include "../someheader.py"
```
