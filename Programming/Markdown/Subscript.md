# Subscript

Some Markdown processors allows to use of **Subscript** and **Superscript**.

To create a **subscript**, use one tilde symbol (~) before and after the
character.
```md
H~2~O
```

### Rendered output

H~2~O

### HTML syntax

Alternatively, if the Markdown application supports HTML:
```md
H<sub>2</sub>O
```
