# PowerShell

[PowerShell](https://en.wikipedia.org/wiki/PowerShell)
is a shell program developed by [[Microsoft|Microsoft]]
for task automation and configuration management

[PowerShell](https://learn.microsoft.com/en-us/powershell/scripting/overview?view=powershell-7.5&viewFallbackFrom=powershell-7.1)
is a cross-platform task automation solution
made up of a command-line shell, a scripting language,
and a configuration management framework.

**PowerShell** runs on Windows, Linux, and macOS.
