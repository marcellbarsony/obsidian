# Ruby Gems

[RubyGems](https://rubygems.org/) is a package manager
for the [[Ruby/General|Ruby]] programming language
that provides a standard format
for distributing Ruby programs and libraries
(*in a self-contained format called a "gem"*),
a tool designed to easily manage the installation of gems,
and a server for distributing them

___

## Gems Install

Install a Ruby Gem package

```sh
gem install <package>
```
