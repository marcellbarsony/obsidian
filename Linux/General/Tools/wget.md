# Wget

[GNU Wget](https://www.gnu.org/software/wget/)
is a program that retrieves content from web servers

___

## Usage

Download a file from the web root

```sh
wget <target>/<file> -O <out_file>
```

<!-- Example {{{-->
> [!example]-
>
> ```sh
> wget 10.129.59.165/flag.txt -O flag.txt
> ```
<!-- }}} -->
