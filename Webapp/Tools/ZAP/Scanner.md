## Active Scan

[Active Scan](https://www.zaproxy.org/docs/desktop/start/features/ascan/)
attempts to find potential vulnerabilities by using known attacks
against the selected targets

> [!danger]
>
> Do not perform an active scan without authorization

1. Launch Active Scan

```sh
CTRL + ALT + A
```

```sh
Tools > Active Scan
```

```sh
CONTEXT > Attack > Active Scan
```
