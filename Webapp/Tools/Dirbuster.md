# Dirbuster

[DirBuster](https://dirbuster.com) is an OWASP-backed security tool
that uncovers hidden directories and files on web servers

## Install

[Kali Tools](https://www.kali.org/tools/dirbuster/) Install

```sh
sudo apt install dirbuster
```
