# Scanner

[Burp Scanner](https://portswigger.net/burp/documentation/scanner)
is an automated dynamic application security testing (*DAST*)
web vulnerability scanner

- [Crawling](https://portswigger.net/burp/documentation/scanner/crawling) -
  The scanner catalogs the content of the application
  and the navigational paths within it:
  it follows links, submits forms, and logs data
  of the application's content

- [Auditing](https://portswigger.net/burp/documentation/scanner/auditing) -
  The scanner analyzes the application's traffic
  and behavior to identify security vulnerabilities and other issues

> [!warning] PRO FEATURE
