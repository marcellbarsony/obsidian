# Web Server Root

## OS

OS default web server roots

| OS      | Path                     |
| ------- | ------------------------ |
| Linux   | `/var/www/html`          |
| Windows | `C:\inetpub\wwwroot`     |

## Web Server

| Web Server | Default Webroot          |
| ---------- | ------------------------ |
| Apache     | `/var/www/html/`         |
| Nginx      | `/usr/local/nginx/html/` |
| IIS        | `c:\inetpub\wwwroot\`    |
| XAMPP      | `C:\xampp\htdocs\`       |
