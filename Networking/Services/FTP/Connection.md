# FTP Connection

## FTP

Synopsis

```sh
ftp <ip> [port]
```

## LFTP

Connect using [lftp](https://linux.die.net/man/1/lftp)
```sh
lftp <ip>
```

## Web browser

Access the FTP server using a web browser

```sh
ftp://<username>:<password>@<ip>
```
