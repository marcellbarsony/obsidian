# General

**Microsoft Internet Information Services**
(*[Microsoft IIS](https://en.wikipedia.org/wiki/Internet_Information_Services)*)
is an extensible web server created by Microsoft
for use with the Windows NT family.

**Microsoft IIS** supports HTTP, HTTP/2, HTTP/3, HTTPS,
[[FTP/General|FTP]], FTPS,
[[SMTP/General|SMTP]] and NNTP
