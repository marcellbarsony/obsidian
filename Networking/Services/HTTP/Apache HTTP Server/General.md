# Apache HTTP Server

[Apache HTTP Server](https://httpd.apache.org/) (*httpd*)
Project is an effort to develop and maintain an open-source HTTP server
for modern operating systems including UNIX and Windows

The goal of this project is to provide a secure,
efficient and extensible server that provides HTTP services
in sync with the current HTTP standards
