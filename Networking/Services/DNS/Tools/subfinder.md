# subfinder

[subfinder](https://github.com/projectdiscovery/subfinder)
is a passive subdomain enumeration tool using passive online resources

## Installation

Install on Debian-based distributions

```sh
sudo apt install subfinder
```

Verify installation

```sh
subfinder -h
```
